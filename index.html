Bibliografia

Detallamos de que se trata Duit y la metodologia para su desarrollo.

Idea

El concepto de hacer un calendario surge en un inicio de la falta de alternativas lo suficientemente accesibles para que dos o más personas puedan accerder a un mismo servicio de forma rapida y eficiente, anteriormente probamos herramientas como notion o google calendar, pero el concepto de poder abrir una pestaña de navegador y ver automaticamente las diferentes tareas y entregas futuras con poner solo un link nos llamo bastante la atención.

Lenguajes utilizados

Los lenguajes que se han usado para este proyecto han sido:
HTML para toda el exoesqueleto de la pagina web.
CSS En los estilos y la decoración de principalmente esta pagina de Bibliografia
JavaScript Para todo el funcionamiento interno del calendario.
